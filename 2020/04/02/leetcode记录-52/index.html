n 皇后问题研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。
给定一个整数 n，返回 n 皇后不同的解决方案的数量

189

题目

给定一个数组，将数组中的元素向右移动 k 个位置，其中 k 是非负数。

示例 1:

输入: [1,2,3,4,5,6,7] 和 k = 3
输出: [5,6,7,1,2,3,4]
解释:
向右旋转 1 步: [7,1,2,3,4,5,6]
向右旋转 2 步: [6,7,1,2,3,4,5]
向右旋转 3 步: [5,6,7,1,2,3,4]

解法1：原地一步一步

  • 时间复杂度：O(n*k)O(n∗k) 。每个元素都被移动 1 步（O(n)O(n)） k次（O(k)O(k)） 。
  • 空间复杂度：O(1)O(1) 。没有额外空间被使用。

解法2：用一个新的数组存放

  • 还是旋转了k次，只是每次旋转后都是把元素放入新的数组。
  • 接下来进入官方的机智解法。以下为官方的真·解法2
  • 时间复杂度： O(n)O(n) 。将数字放到新的数组中需要一遍遍历，另一边来把新数组的元素拷贝回原数组。
  • 空间复杂度： O(n)O(n)。另一个数组需要原数组长度的空间。

解法4：反转

我们要的结果：k%n个（不是k个，因为k有可能比数组的长度还长）尾部元素会被移动到头部，剩下的元素会被向后移动。
步骤可为：

  1. 将所有元素反转（此时所有数字都为倒序）
  2. 将前k%n个元素反转（此时前k个是顺序，后面的是倒序）
  3. 将后l-k%n个元素反转

官方的简洁反转如下：

  • 时间复杂度：O(n)O(n) 。 nn 个元素被反转了总共 3 次。
  • 空间复杂度：O(1)O(1) 。 没有使用额外的空间。
