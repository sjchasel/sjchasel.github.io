Deep Keyphrase Generation

Summary

We summarize a text according to its content. But existing approaches just based on features such as frequency of the word, ignoring real meaning of a text’s content. So author utilize an encoder-decoder framework based on RNN and introduce copy mechanism. This way you can generate words that do not exist in the source text and improve the effect of key phrase generation.

Research Objective

Propose a new keyphrases generation model which can predict absent keyphrases and can utilize syntactic features to achieve the effect of understanding the semantics of the text.

Problem Statement

Most of the existing keyphrase extraction algorithms follow this two steps:

  1. acquire a list of keyphrase candidates
  2. rank the candidates

But these approaches have two problems:

  1. Cannot generate keyphrases that are not in the source text
  2. Keyphrases are generated only according to text structure(Such as which two words often appear together), which cannot reveal the semantics of the text

    Method(s)

    encoder

    Encoder compresses the source text to a hidden representation.

    decoder

    Decoder generate keyphrases form hidden representation.

    attention mechanism

    Use attention mechanism to acquire context vector c.

    copy mechanism

    Copy mechanism can make model generate absent keyphrases.

    Evaluation

    datasets

    Training Dataset

    author crawled 567830 articles and used 527830 articles of them to train model.

    Testing Dataset

    KP20k(20000 articles of them) and other datasets

    baseline

    Tf-Idf、TextRank、SingleRank、ExpandRank

    Evaluation Metric

  • macro-averaged precision
  • recall
  • F-measure (F1)

    Conclusion

    CopyRNN can performs well on predicting present keyphrases and can generate absent keyphrases.

Single Document Keyphrase Extraction Using Neighborhood Knowledge

Summary

The author propose a new algorithm called ExpandRank which does not extract keywords based on information from single articles, but rather takes into account other articles in the same domain. Efficiency of keyphrases extraction has increased because articles in the same domain can provide more information for the target document.

Research Objective

In order to improve the efficiency of keyphrase extraction, texts in the same domain should be considered when extracting keyphrases from a single text.

Background and Problems

Existing keyphrase extraction methods are based on the single content of a single article. But in fact, multiple articles in the same domain can corroborate each other and provide more information about each other’s keyphrases.
