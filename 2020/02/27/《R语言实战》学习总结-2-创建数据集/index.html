数据结构

一些概念简介

数据集

数据集通常是由数据构成的一个矩形数组，行表示观测，列表示变量。
只要是数据都可以说是数据集。

对象

对象是可以赋值给变量的任何事物，包括变量、数据结构、函数、图形等。
对象都有某种模式，描述此对象如何存储、如何处理。

数据框

数据框是用于存储数据结构的一种结构：列表示变量，行表示观测。
同一个数据框中可以存储不同类型的变量。
数据框是数据集的一种。

因子

是名义型变量或有序型变量。

向量

  • 向量是存储数值类型、字符型或逻辑型数据的一维数组。
  • 单个向量中的数据必须拥有相同的类型或模式。
  • 标量是只含一个元素的向量。
  • 用函数c()创建向量
  • 访问向量中的元素

    矩阵

    介绍

  • 矩阵是个二维数组，每个元素都有相同的模式。
  • 通过函数matrix创建
  • 参数解释
    • vector包含矩阵的元素
    • nrow与ncol制定行和列的维数
    • dimnames包含了可选的、以字符型向量表示的行名和列名
    • byrow=TRUE表示矩阵按行填充，FALSE表示按列填充

    因子

    概念

变量

  1. 名义型变量
    是没有顺序之分的类别变量（感觉像统计里的定类尺度？）。比如糖尿病类型（Type1，Type2）。
  2. 有序型变量
    表示一种顺序关系，而非数量关系（像定序尺度）。比如病情Status(poor,improved,excellent)，我们可以将它们排序，但不知道之间定量地相差了多少。
  3. 连续型变量
    连续型变量可以呈现为某个范围内的任意值，并同时表示了顺序和数量。比如年龄，又连续，又可以进行数量上的比较。

因子
包括名义型变量和有序型变量，把它们单独拎出来命名，因为它们很重要。因子决定了数据的分析方式以及如何进行视觉呈现。

唯一值是所有行数据中，限定唯一值的列（或者列组合），不能出现重复数据。

    处理对象的实用函数

    函数 用途
    length(object) 显示对象中元素/成分的数量
    dim(object) 显示某个对象的维度
    str(object) 显示某个对象的结构
    class(object) 显示某个对象的类或类型
    mode(object) 显示某个对象的模式
    names(object) 显示某对象中各成分的名称
    c(object, object,…) 将对象合并入一个向量
    cbind(object, object, …) 按列合并对象
    rbind(object, object, …) 按行合并对象
    Object 输出某个对象
    head(object) 列出某个对象的开始部分
    tail(object) 列出某个对象的最后部分
    ls() 显示当前的对象列表
    rm(object, object, …) 删除一个或更多个对象。语句rm(list = ls()) 将删除当前工作环境中的几乎所有对象*
    newobject <- edit(object) 编辑对象并另存为newobject
    fix(object) 直接编辑对象

程序员的注意事项

  • 对象名称中的句点（.）没有特殊意义，R中的$与其他语言中的句点有类似的含义：指定一个对象中的某些部分
  • R没有多行注释功能，非要写一大段的话可以写在if(FALSE){}中
  • R中没有标量（没有方向的量）。标量以单元素向量的形式出现
  • R的下标从1开始
  • 变量不需要声明，它们在首次被赋值时生成
  • 将一个值赋给某个向量（或矩阵、数组、列表）中的一个不存在的元素时，R会自己加上。
